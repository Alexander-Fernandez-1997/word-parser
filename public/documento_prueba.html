Documento de Prueba con Múltiples Formatos

Este es un documento que contiene diferentes estilos y formatos para probar la aplicación de limpieza de formato de Word. Veremos cómo se comporta cuando encuentra diferentes elementos de formato.

Sección 1: Diferentes Estilos de Texto

Este párrafo contiene texto en negrita, texto en cursiva, texto subrayado y texto resaltado.

También podemos tener diferentes tamaños de texto, incluyendo texto pequeño.

Y por supuesto, texto con diferentes colores como azul y verde.

Este párrafo tiene justificación completa. Se añade texto adicional para que se pueda apreciar correctamente el efecto de la justificación en múltiples líneas de texto. La justificación hace que el texto se estire para ocupar todo el ancho disponible, creando márgenes rectos tanto a la izquierda como a la derecha del bloque de texto.
